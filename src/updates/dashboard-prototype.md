The public [evidence dashboard](/evidence-dashboard/) is now live on the site.

The dashboard shows the intended layout and structure for how owner-submitted data will be
presented, including:

- Summary statistics (owners contributed, vehicles with HV work, SoH averages, recall rates)
- Bar charts for SoH distribution, model year distribution, recall status, loan car provision
- Warranty status breakdown (inside vs outside 8-year / 100,000-mile battery warranty)

All figures are currently illustrative — real data collection has not yet begun.
